# Universal optimizer application

Application `Universal optimizer` contains various optimization algorithms for different problems. Application is developed in `Python` programming language, using `poetry` tool for project management, `unittest`  library for unit testing and `Sphinx` for documentation generation.

Packages required for successful work of the application are specified in the file `requirements.txt` and can be installed in the following way:

```bash
pip install -r requirements.txt
```

Application is based on `Universal optimizer library`, which is, during development, stored within `lib` folder.

Prior to development, this library should be installed in editable manner:

```bash
pip install -e lib
```
